# smartdorm-management-system

Hệ thống quản lý ký túc xá (SmartDorm) – Fullstack: React, Node.js, Express, MongoDB.

## Cấu trúc repo

- **`smartdorm-backend/`** – API backend (Express, MongoDB, JWT, thông báo, …)
- **`smartdorm-frontend/`** – Ứng dụng web React (TypeScript)

## Chạy dự án

### Backend

```bash
cd smartdorm-backend
cp .env.example .env   # chỉnh .env theo môi trường
npm install
npm run dev
```

### Frontend

```bash
cd smartdorm-frontend
npm install
npm run dev
```

Xem thêm hướng dẫn trong từng thư mục.
